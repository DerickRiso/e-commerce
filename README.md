# E-commerce Furniro

Projeto que implementa um e-commerce de móveis usando:
- React, Typescript, React Routes, CSS e Vite para o frontend;
- NestJS, Class validator, TypeORM para conexão com banco de dados PostgreSQL;
- Foi utilizado Git flow por meio de branchs e pull-requests para controle de versão do projeto.

## Funcionalidades

1. Página inicial com prévia dos produtos
2. Página de produto único direcionada diretamente com React Router
3. Lista de produtos com filtro por categoria e ordenação dinâmica
4. Funcionalidade de aplicação de desconto por meio do banco de dados
